# MAUS | Macro Automation Utility Suite

> A lightweight, fast automation utility for Windows. Built by [MRB Labs](https://github.com/MrBobertus)

[![Download](https://img.shields.io/badge/Download-maus.exe-red?style=for-the-badge\&logo=windows)](https://github.com/MrBobertus/maus/raw/refs/heads/main/bin/Release/maus.exe)

---

## Features

* **Mouse automation** - automate left, middle, or right mouse clicks
* **Keyboard automation** - repeatedly press any keyboard key
* **Adjustable interval** - set hours, minutes, seconds, and milliseconds
* **Single or double click** - choose how mouse clicks are performed
* **Repeat mode** - run indefinitely or stop after a specified number of actions
* **Custom hotkey** - bind any key to toggle automation on/off
* **Emergency stop** - press `F6` at any time to immediately stop automation
* **Lightweight** - under 1 MB, no installation required

---

## Hotkeys

| Key                   | Action                     |
| --------------------- | -------------------------- |
| Custom (default `F2`) | Toggle automation on / off |
| `F6`                  | Emergency stop             |
